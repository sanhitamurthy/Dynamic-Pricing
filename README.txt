This is a dynamic pricing project for CS5100 course at NEU. 

To run the project, download both the python files. Change the location of the dataset in both the python files. You can find the dataset at 'https://www.kaggle.com/c/mercari-price-suggestion-challenge/data'

1. Run KNN_classification.py in pycharm  to run KNN classification algorithm on the dataset.
2. Run LightGBM.py in pycharm  to run Linear Regression, Light GBM and Ridge Regression on the dataset.
